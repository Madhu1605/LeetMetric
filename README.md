# LeetMetric

LeetMetric is a web application that visualizes LeetCode user statistics using a public third-party API.

## Features
- Search LeetCode users by username
- Displays problems solved by difficulty (Easy / Medium / Hard)
- Circular progress indicators
- Acceptance rate, ranking, and contribution points
- Supports Enter key for search

## Tech Stack
- HTML
- CSS
- JavaScript

## How It Works
1. User enters a LeetCode username
2. App validates the input
3. Fetches data from a public LeetCode stats API
4. Updates UI dynamically using JavaScript

## Disclaimer
This project uses an unofficial public API and is intended for learning and portfolio purposes only.
